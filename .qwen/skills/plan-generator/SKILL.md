# Plan Generator Skill

Create structured action plans with checkboxes for complex multi-step tasks.

## Purpose

When Qwen Code receives a complex task, it creates a Plan.md file that:
1. Breaks down the task into manageable steps
2. Tracks progress with checkboxes
3. Documents decisions and outcomes
4. Provides audit trail for completed work

## Usage

### Generate Plan from Action File

```bash
python scripts/plan_generator.py \
  --action-file /path/to/Needs_Action/FILE_xxx.md \
  --vault /path/to/AI_Employee_Vault
```

### Generate Plan from Prompt

```bash
python scripts/plan_generator.py \
  --prompt "Process monthly invoices and send reminders for overdue payments" \
  --vault /path/to/AI_Employee_Vault
```

## Features

- ✅ Automatic step generation based on task type
- ✅ Progress tracking (completed/total steps)
- ✅ Decision logging for audit trail
- ✅ Blocker identification
- ✅ Links to related action files
- ✅ Status updates (pending, in_progress, completed, blocked)

## Task Types and Default Steps

### Invoice Processing
1. Read action file
2. Check Company Handbook rules
3. Verify invoice details
4. Check payment status
5. Generate reminder if overdue
6. Create approval request if needed
7. Update accounting records
8. Update Dashboard
9. Move to Done

### Email Response
1. Read email content
2. Check sender in known contacts
3. Determine response type needed
4. Draft response
5. Check if approval required
6. Send email (if approved)
7. Log action
8. Archive email
9. Update Dashboard

### Social Media Post
1. Read post content
2. Check Company Handbook rules
3. Verify content appropriateness
4. Add hashtags
5. Create approval request
6. Schedule/post (if approved)
7. Track engagement
8. Log action
9. Update Dashboard

## Integration with Qwen Code

Qwen Code automatically:
1. Creates Plan.md when task has >3 steps
2. Updates checkboxes as steps complete
3. Moves Plan.md to Done when all steps checked
4. Logs decisions in the Decisions table
